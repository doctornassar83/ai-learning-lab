# AI Learning Lab Guides

This folder contains concise foundational guides for the AI Learning Lab project.

The purpose of these guides is to provide:

- development environment setup
- terminal workflows
- Git and GitHub basics
- Python environment management
- AI engineering fundamentals
- debugging references
- productivity workflows

These are intentionally kept as small Markdown reference files.

Large tutorials, projects, experiments, and notebooks are stored outside this folder in their dedicated project directories.
